# My workshop

Replace this file with your subject content, or rename it and update `metadata.yaml` (`documents`, `project.entrypoint`).

## Getting started

Write your workshop in Markdown. Headings (`#`, `##`, …) drive the **toc** section in `metadata.yaml` (kept in sync by the tooling in [**workshop-metadata-tools**](https://github.com/Manta-Epitech-Academy/workshop-metadata-tools) and CI).

## Next steps

1. Edit `metadata.yaml`: `project`, `runtime`, and list every `.md` file under `documents`.
2. Run `sync_metadata_toc.py`, `generate_readme.py`, and `check_toc.py` locally using the [tools repo](https://github.com/Manta-Epitech-Academy/workshop-metadata-tools) (see **TEMPLATE.md**), then commit `metadata.yaml` and `README.md`.
3. Push to GitHub; the **Verify metadata** workflow checks schema, toc, and competency codes.
